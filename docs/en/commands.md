# Command Overview

Here you will find all the important commands you need for your adventure on HMT-Network.

## General Commands
| Command | Description |
| :--- | :--- |
| `/joinq {server}` | Join a server (e.g., `smp01`). |
| `/leaveq` | Leave the current queue. |
| `/rtp` | Random teleport on the SMP (does not work in lobbies). |
| `/sb toggle` | Toggles the scoreboard on or off. |
| `/msg {player}` | Sends a private message. |
| `/reply` | Quickly reply to the last message. |

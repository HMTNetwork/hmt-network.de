# Command Overview

Here you will find all the essential commands you need for your adventure on the HMT-Network.  
Cooldowns are tiered by rank (Player / Plus / Star).

## General Commands & Teleportation
| Command | Description | Cooldown (Player / Plus / Star) |
| :--- | :--- | :--- |
| `/joinq {server}` | Join a server (e.g., `smp01`). |  |
| `/leaveq` | Leave the current queue. |  |
| `/rtp` | Random teleport into the wilderness. | 60 min / 45 min / 33 min |
| `/home` | Teleport to your set home. | 25 min / 18 min / 14 min |
| `/tpa {player}` | Send a teleport request to a player. | 20 min / 15 min / 11 min |
| `/tpahere {player}` | Ask a player to teleport to you. | 20 min / 15 min / 11 min |
| `/spawn` | Teleport back to the spawn point. | 20 min / 15 min / 11 min |
| `/sb toggle` | Toggles the scoreboard on or off. |  |
| `/msg {player}` | Send a private message. |  |
| `/reply` | Quickly reply to the last message. |  |

# Server Regeln

Um eine angenehme und faire Spielumgebung für alle Spieler zu gewährleisten, gelten auf allen unseren Servern folgende Regeln:

## Allgemeine Verhaltensregeln

### Respektvoller Umgang

- Behandle alle Spieler mit Respekt, unabhängig von ihrem Rang
- Diskriminierung, Beleidigungen oder Hassrede jeglicher Art sind nicht erlaubt
- Spamming, Trollen oder absichtliche Provokation anderer Spieler ist untersagt

### Cheating und Hacking

- Das Verwenden von Hacks, Cheats oder vorteilhaften Mods ist strengstens verboten
- Exploits dürfen nicht ausgenutzt werden
- Bugs müssen gemeldet werden, nicht ausgenutzt werden
- X-Ray Texture Packs sind nicht erlaubt

### Fair Play

- Griefing anderer Spieler ist nicht erlaubt
- Das Zerstören oder Beschädigen von fremden Bauwerken ohne Erlaubnis ist verboten
- Das Blockieren von Spawn-Punkten oder wichtigen Bereichen ist untersagt
- Das Farmen von Ressourcen in geschützten Gebieten ist nicht erlaubt

## Chat-Regeln

### Kommunikation

- Kommuniziere auf Deutsch oder Englisch
- Werbung für andere Server oder Communities ist nicht erlaubt
- Links zu externen Websites müssen vorher mit einem Moderator abgestimmt werden
- Flaming oder Streitgespräche sollten vermieden werden

### Privatsphäre

- Persönliche Daten anderer Spieler dürfen nicht ohne Zustimmung veröffentlicht werden
- Real-Life-Threats oder Drohungen jeglicher Art sind absolut verboten

## Server-spezifische Regeln

### Survival Server

- PvE (Player versus Environment) ist erlaubt
- PvP (Player versus Player) ist nur erlaubt, wenn beide Spieler PvP aktiviert haben
- Das Claimen von Gebieten muss den Richtlinien entsprechen

### Hardcore Server

- PvE (Player versus Environment) ist erlaubt
- PvP (Player versus Player) ist nur erlaubt, wenn beide Spieler PvP aktiviert haben
- Das Claimen von Gebieten muss den Richtlinien entsprechen
- Wenn man stirbt erhält man keinen erneuten Zugriff zum Server

## Konsequenzen

Verstöße gegen diese Regeln können je nach Schwere der Tat zu folgenden Konsequenzen führen:

1. Verwarnung (Warn)
2. Temporärer Kick vom Server
3. Temporärer Bann vom Server
4. Permanenter Bann vom Server

Bei schwerwiegenden Verstößen kann sofort ein permanenter Bann erfolgen.

## Melden von Regelverstößen

Regelverstöße können folgendermaßen gemeldet werden:

- Über den `/report` Command auf dem Discord-Server
- Über das Ticket-System auf unserem Discord-Server

## Änderungen der Regeln

Diese Regeln können jederzeit aktualisiert werden. Spieler werden über wesentliche Änderungen informiert. Es liegt in der Verantwortung jedes Spielers, sich über die aktuellen Regeln zu informieren.

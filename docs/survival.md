# Survival Server

Der klassische Minecraft-Spaß - Baue, erkunde und überlebe in einer entspannten Atmosphäre!

## Über diesen Server

Der Survival-Server ist perfekt für Spieler, die Minecraft in seiner klassischen Form genießen möchten. Hier kannst du in Ruhe bauen, erforschen und mit anderen zusammenarbeiten:

- Standard-Schwierigkeitsgrad für angenehmes Spielerlebnis
- Keine Permadeath - Du kannst dich aus Fehlern erholen
- Regelmäßige Backups für deine Fortschritte
- Faire PvP-Regeln und klare Grenzen

Egal ob Anfänger oder erfahrener Spieler - auf diesem Server findet jeder seinen Platz zum Entwickeln und Genießen!

## Features

### Entspanntes Spielvergnügen

Erkunde die Welt ohne Druck - baue dein Traumhaus oder überlebe abenteuerlich.

### Community Projekte

Beteilige dich an gemeinsamen Bauprojekten und stärke die Gemeinschaft.

### Regelmäßige Events

Nimm an Bauwettbewerben, Abenteuern und speziellen Herausforderungen teil.

### Sichere Umgebung

Gerechte Regeln und Moderation sorgen für ein faires Spielerlebnis.

## Beitritt

Um dem Survival Server beizutreten, verwende einfach die Serveradresse `mc.hmt-network.de` in deinem Minecraft Client.

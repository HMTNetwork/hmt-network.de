# Befehls-Übersicht

Hier findest du alle wichtigen Commands, die du für dein Abenteuer auf HMT-Network benötigst. 
Die Cooldowns sind nach Rängen gestaffelt (Spieler / Plus / Star).

## Allgemeine Befehle & Teleportation
| Command | Beschreibung | Cooldown (Spieler / Plus / Star) |
| :--- | :--- | :--- |
| `/joinq {server}` | Tritt einem Server bei (z.B. `smp01`). |  |
| `/leaveq` | Verlasse die aktuelle Warteschlange. |  |
| `/rtp` | Zufälliger Teleport in die Wildnis. | 60 Min / 45 Min / 33 Min |
| `/home` | Teleportiert dich zu deinem Zuhause. | 25 Min / 18 Min / 14 Min |
| `/tpa {spieler}` | Sendet eine Teleportanfrage an einen Spieler. | 20 Min / 15 Min / 11 Min |
| `/tpahere {spieler}` | Bittet einen Spieler, sich zu dir zu teleportieren. | 20 Min / 15 Min / 11 Min |
| `/spawn` | Teleportiert dich zurück zum Startpunkt. | 20 Min / 15 Min / 11 Min |
| `/sb toggle` | Schaltet das Scoreboard an oder aus. |  |
| `/msg {spieler}` | Sendet eine private Nachricht. |  |
| `/reply` | Antworte schnell auf die letzte Nachricht. |  |

## Wirtschaft & Management
| Command | Beschreibung | Cooldown / Info |
| :--- | :--- | :--- |
| `/daily` | Fordere deine tägliche Belohnung ein. | Alle 24 Stunden |
| `/kit tools` | Erhalte deine Starter-Ausrüstung. | 3 Std / 2,2 Std / 1,7 Std |
| `/shop` | Öffnet den Shop für Totems, Kristalle etc. |  |
| `/ah` | Öffnet das Auktionshaus. |  |
| `/crates` oder `/cc` | Zugriff auf das Crate-System. |  |
| `/balance` | Zeigt dein aktuelles Guthaben in € an. |  |
| `/pay {spieler} {betrag}` | Überweist Geld an einen anderen Spieler. |  |
| `/team` | Menü zum Erstellen und Verwalten von Gilden. |  |

---
**Hinweis:** Teleport-Befehle werden abgebrochen, wenn du dich während des Warmups bewegst!

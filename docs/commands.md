# Befehls-Übersicht

Hier findest du alle wichtigen Commands, die du für dein Abenteuer auf HMT-Network benötigst.

## Allgemeine Befehle
| Command | Beschreibung |
| :--- | :--- |
| `/joinq {server}` | Tritt einem Server bei (z.B. `smp01`). |
| `/leaveq` | Verlasse die aktuelle Warteschlange. |
| `/rtp` | Zufälliger Teleport auf dem SMP (funktioniert nicht in Lobbies). |
| `/sb toggle` | Schaltet das Scoreboard an oder aus. |
| `/msg {spieler}` | Sendet eine private Nachricht. |
| `/reply` | Antworte schnell auf die letzte Nachricht. |

## Wirtschaft & Management
| Command | Beschreibung |
| :--- | :--- |
| `/shop` | Öffnet den Shop für Totems, Kristalle etc. |
| `/ah` | Öffnet das Auktionshaus. |
| `/crates` oder `/cc` | Zugriff auf das Crate-System. |
| `/balance` | Zeigt dein aktuelles Guthaben in € an. |
| `/pay {spieler} {betrag}` | Überweist Geld an einen anderen Spieler. |
| `/team` | Menü zum Erstellen und Verwalten von Gilden. |

# Server Commands

Eine Übersicht der verfügbaren Befehle auf unseren Minecraft-Servern.

## Allgemeine Befehle

### Hilfe und Informationen

- `/help` - Zeigt eine Liste aller verfügbaren Befehle an
- `/spawn` - Teleportiert dich zum Spawn-Punkt
- `/home` - Teleportiert dich zu deinem Home-Punkt
- `/sethome` - Setzt deinen Home-Punkt an deiner aktuellen Position

### Soziale Interaktion

- `/msg <Spieler> <Nachricht>` - Sendet eine private Nachricht an einen Spieler
- `/tell` - Alternative zu `/msg`

### Teleportation

- `/tpa <Spieler>` - Sendet eine Teleport-Anfrage an einen Spieler
- `/tpaccept` - Akzeptiert eine eingehende Teleport-Anfrage
- `/tpdeny` - Lehnt eine eingehende Teleport-Anfrage ab
- `/tpahere <Spieler>` - Sendet eine Teleport-Anfrage an einen Spieler, teleportiert aber zu dir

## Survival Server Befehle

### Regionen und Schutz

**Diese Commands sind vorübergehend deaktiviert!**

- `/xclaim claim <Name>` - Beansprucht ein Gebiet um dich herum
- `/xclaim info` - Zeigt Informationen über die Region an, in der du dich befindest
- `/xclaim trust <Spieler>` - Erlaubt einem Spieler Zugang zu deiner Region

## Hardcore Server Befehle

Aufgrund der Natur des Hardcore-Servers sind einige Befehle eingeschränkt oder nicht verfügbar, um das Überlebensgefühl zu erhalten.

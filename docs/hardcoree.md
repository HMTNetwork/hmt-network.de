# Hardcore Server

Die ultimative Überlebensherausforderung - Wenn du bereit bist, das Risiko einzugehen!

## Über diesen Server

Der Hardcore-Server ist für Spieler gedacht, die das ultimative Überlebensgefühl suchen. Hier gelten besondere Regeln:

- Nur ein Leben - Tod bedeutet permanenter Verlust
- Schwierigkeitsgrad ist auf "Hardcore" eingestellt
- Besondere PvP-Regeln gelten
- Gemeinschaftliches Überleben wird belohnt

Diese Herausforderung ist nichts für schwache Nerven. Nur die Tapfersten überleben lange auf diesem Server!

## Features

### Permadeath

Jeder Tod ist permanent. Sei vorsichtig und spiele klug!

### Hardcore Difficulty

Maximale Herausforderung mit erhöhtem Schaden und Hunger.

### Spezielle Events

Regelmäßige Events und Herausforderungen für alle Spieler.

### Community Fokus

Zusammenarbeit ist der Schlüssel zum Überleben.

## Beitritt

Um dem Hardcore Server beizutreten, verwende einfach die Serveradresse `mc.hmt-network.de` in deinem Minecraft Client. Beachte jedoch, dass dieser Server besondere Risiken birgt - sei gewarnt!

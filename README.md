# HMT Network Website

Willkommen beim offiziellen Repository der HMT Network Minecraft-Server Webseite.

## Über das Projekt

Diese Website dient als zentrale Anlaufstelle für alle Informationen rund um die HMT Network Minecraft-Server. Sie bietet Spielern einen Überblick über unsere Server, Community und Regeln.

## Server

Wir betreiben zwei verschiedene Minecraft-Server:

- **Survival Server**: Klassisches Minecraft-Erlebnis mit entspanntem Gameplay
- **Hardcore Server**: Herausforderndes Erlebnis mit Permadeath und erhöhter Schwierigkeit

## Technologie

Die Website ist eine statische HTML-Seite, die mit folgenden Technologien erstellt wurde:

![Technologie-Stack](https://skillicons.dev/icons?i=html,css,javascript,nginx,git,github,vscode)

- **HTML5** - Struktur der Webseite
- **CSS3** - Gestaltung und Layout
- **JavaScript** - Interaktive Elemente
- **Nginx** - Webserver
- **Docsify** - Dokumentationssystem

## Struktur

```
.
├── about/                 # Über uns Bereich
│   └── team/             # Team Mitglieder
├── assets/               # Statische Ressourcen
│   ├── css/              # Stylesheets
│   ├── img/              # Bilder und Logos
│   ├── js/               # JavaScript Dateien
│   └── templates/        # HTML Templates
├── docs/                 # Dokumentation
├── servers/              # Server spezifische Seiten
│   ├── hardcore/
│   └── survival/
├── index.html            # Hauptseite
└── nginx.conf           # Nginx Konfiguration
```

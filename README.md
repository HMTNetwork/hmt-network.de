# HMT Network Website

Willkommen beim offiziellen Repository der HMT Network Minecraft-Server Webseite.

## Über das Projekt

Diese Website dient als zentrale Anlaufstelle für alle Informationen rund um die HMT Network Minecraft-Server. Sie bietet Spielern einen Überblick über unsere Server, Community und Regeln sowie aktuelle Blog-Beiträge und Ankündigungen.

## Server

Wir betreiben mehrere Minecraft-Server:

- **Lobby Server**: Zentraler Treffpunkt unserer Community
- **Survival Server**: Klassisches Minecraft-Erlebnis mit entspanntem Gameplay
- **Hardcore Server**: Herausforderndes Erlebnis mit Permadeath und erhöhter Schwierigkeit

## Technologie

Die Website ist eine moderne statische HTML-Seite, die mit folgenden Technologien erstellt wurde:

![Technologie-Stack](https://skillicons.dev/icons?i=html,css,javascript,nginx,git,github,vscode)

- **HTML5** - Struktur der Webseite
- **CSS3** - Gestaltung und Layout mit benutzerdefiniertem Design-System
- **JavaScript** - Interaktive Elemente und dynamische Inhalte
- **Lucide Icons** - Moderne SVG-Symboldesigns
- **Google Fonts** - Schriftarten (Inter und JetBrains Mono)
- **Nginx** - Webserver und Reverse Proxy
- **Docsify** - Dokumentationssystem

## Struktur

```
.
├── about/                 # Über uns Bereich
│   └── team/              # Team Mitglieder
├── assets/                # Statische Ressourcen
│   ├── css/               # Stylesheets
│   ├── img/               # Bilder und Logos
│   ├── js/                # JavaScript Dateien
│   └── templates/         # HTML Templates
├── blog/                  # Blog Beiträge
├── branding/              # Branding Richtlinien
├── docs/                  # Dokumentation
├── security/              # Sicherheitsinformationen
├── servers/               # Server spezifische Seiten
│   ├── hardcore/
│   └── survival/

├── 404.html               # Fehlerseite
├── nginx.conf             # Nginx Konfiguration
├── redirects.conf         # Weiterleitungsregeln
├── robots.txt             # Suchmaschinen-Richtlinien
└── sitemap.xml            # Sitemap für Suchmaschinen
```
